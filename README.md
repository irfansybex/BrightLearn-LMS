# LMS Project

**Advanced Learning Management System** built with .NET Core and Blazor.

## Features
- Student, Teacher & Admin portals
- Course management, quizzes, and assignments
- Certificate generation (PDF)
- Attendance tracking
- Progress dashboards
- Secure authentication with JWT + Role-based access
- Notifications & Email alerts

## Tech Stack
- Backend: .NET Core, EF Core
- Frontend: Blazor, React
- Database: SQL Server
- Tools: Azure, Docker, Bootstrap
